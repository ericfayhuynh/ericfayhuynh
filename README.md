## 🙋‍♂️ About

Hey — I’m Eric.

I work pretty hands-on with data day-to-day, mostly in Python + MySQL. A lot of what I do is maintaining and making sense of large contact datasets — figuring out where data comes from, cleaning it up, and making it usable for analytics and campaigns.

It’s a lot of fine-tuning every day. Some things break, some things work, and a big part of the job is just making systems more reliable over time.

I’ve worked on building out a contact database system from scratch, which was one of the more fun projects I’ve done, and now I spend a lot of time maintaining and improving it. That includes handling unsubscribe logic, tracking data sources, and helping support conversion analytics.

I also helped transition a lot of workflows from Excel into SQL (MySQL, DBeaver, CLI), which made things way more scalable and easier to manage.

Still figuring out exactly what I like most, but I’ve been leaning more toward data engineering and systems work as I get more exposure to larger datasets.
